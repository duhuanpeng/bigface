### 功能计划

1. 时钟，准备用美信那个大块头，自带电池的，插件式。
2. 红外遥控接收头
3. 红外串口，当加上外壳后，可以用来编程或通信。
4. 串口接口
5. 温度传感器
6. 湿度传感器
7. 气压传感器

### 元件选型
1. 主控单片机 DS89C450ENL
2. 字库 flash 4MB, 8PIN
3. USB<->uart 芯片
4. 时钟芯片，自带电池晶振 DS17887，8位并口接口
5. 时钟芯片2，DS1302，小体积
6. DS1624 温度传感芯片I2C接口，带有EEPROM
7. DS1624 带有EEPROM
8. 光传感器/封装过小，考虑中
9. 三轴传感器/封装过小，考虑中
10. 心跳 LED
11. 蜂鸣器
12. 麦克风
13. 继电器板
14. 红外遥控
15. 红外串口
